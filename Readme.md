# IP Logger

Tool to locate IP.

![](https://i.imgur.com/k2GPpGK.png)

## Linux
### Instalation
```
git clone github.com/MomboteQ/IP-Locator
cd IP-Locator
pip3 install -r requirements.txt
```

### Using
```
python3 IP-Locator.py
```

## Windows
### Instalation
```
git clone github.com/MomboteQ/IP-Locator
cd IP-Locator
pip install -r requirements.txt
```

## Using
```
python IP-Locator.py
```
